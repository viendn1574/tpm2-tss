One-Call function for Fapi_Unseal

Unseals data from a seal in the FAPI metadata store.

Parameters

[in]pathThe path to the sealed data
[out]dataThe decrypted data after unsealing. May be NULL
[out]sizeThe size of data in bytes. May be NULL
Return values

TSS2_FAPI_RC_BAD_REFERENCEif context or path is NULL.

TSS2_FAPI_RC_IO_ERRORif the data cannot be saved.

TSS2_FAPI_RC_PATH_NOT_FOUNDif a FAPI object path was not found during authorization.
